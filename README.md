# Stackly Restaurant Website

Premium restaurant website built with **HTML, CSS, and JavaScript only**.

## Open the site

1. Open `index.html` in your browser (double-click or use Live Server).
2. The loading screen appears first, then the home page.

## Pages

| Page | File |
|------|------|
| Home | `index.html` |
| About | `about.html` |
| Menu | `menu.html` |
| Services | `services.html` |
| Contact | `contact.html` |
| Login | `login.html` |
| Client Dashboard | `client-dashboard.html` |
| Admin Dashboard | `admin-dashboard.html` |

## Login (demo)

- **Client:** any email/password → Client Dashboard
- **Admin:** any credentials → Admin Dashboard

Use the tabs on the login page to switch between Client and Admin portals.

## Features

- Responsive from **320px** upward
- Shared navigation & footer (injected via JS)
- 10+ sections per page
- Animated loading screen
- Scroll reveal, counters, particles, FAQ accordions
- Dashboards with sidebar navigation (scrolls to sections)
- Canvas charts (line, bar, donut)

## Domain note

Branded for **reastaurent** / **Stackly** — update footer links when deploying to your host.
